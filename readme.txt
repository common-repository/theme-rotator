== Description ==

Theme Rotater lets you define dates where your themes should be activated automatically, e.g Ramadan theme, Halloween theme, Christmas theme.. etc.

You can choose from the current themes, and set a date when the theme should be activated automatically.

== Installation ==

1. Upload `theme-rotator.zip` to the `/wp-content/plugins/` directory
2. Extract the uploaded archive
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Open plugin options page from within Appearance menu
5. Start creating rules!

== Frequently Asked Questions ==

= What if i want to activate the theme for only couple of days ? =

Create a rule for the start date of the theme, and then a second rule to activate your default theme on the end date you wish.

= Where did my previous version settings go ? =

v0.5 does not keep the settings from previous releases, due to total rewrite of the code for better performance.
